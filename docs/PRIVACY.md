# 隐私说明

小红书图文素材助手不会收集、上传或出售任何用户数据。

## 数据处理

- 扩展只读取当前打开的小红书帖子页面中的图片地址、可直接下载的视频地址、标题、正文和原帖链接。
- 扩展只在用户点击保存时保存用户选择的图片、视频和当前可见页面截图。
- 当用户选择“只保存图片”时，扩展不会保存视频、正文、原帖链接或截图。
- 扩展不会把图片地址、视频地址、页面内容、截图或用户信息发送到第三方服务器。
- 如果用户选择了自定义保存文件夹，浏览器可能会保存该文件夹授权，以便后续继续使用。
- 扩展会在本地保存用户上次选择的保存模式，以及默认下载文件夹下素材包名称的自动编号计数。

## 权限说明

- `activeTab`：访问当前激活的小红书页面。
- `downloads`：把图片保存到浏览器下载目录。
- `scripting`：在当前页面注入识别图片所需脚本。
- `storage`：在本地保存用户偏好和自动编号计数。
- `tabs`：判断当前页面是否为小红书页面。
- `host_permissions`：允许扩展在小红书和小红书图片 CDN 页面中识别图片。
