# 小红书图文素材助手

这是一个 Chrome / Edge 浏览器扩展，用来把当前打开的小红书帖子整理成素材包。

## 发布目录

插件源码在项目根目录的 `extension/` 文件夹中。发布、加载未打包扩展、制作压缩包时都使用这个文件夹。

## 主要功能

- 识别当前帖子里的图片和可直接下载的 MP4 视频。
- 默认全选素材，支持单张选择或取消选择。
- 支持“保存帖子”和“只保存图片”两种模式。
- 保存完整帖子时包含图片、视频、正文、原帖链接和截图。
- 保存路径默认是浏览器下载目录；用户选择其他文件夹后会记住。
- 同名素材包自动编号，避免覆盖。

## 素材包结构

```text
帖子标题/
  images/
    01.jpg
    02.jpg
  videos/
    01.mp4
  正文.txt
  原帖链接.html
  截图.png
```

选择“只保存图片”时，只保存 `images/` 文件夹。
