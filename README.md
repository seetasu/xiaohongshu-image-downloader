# 小红书图文素材助手

一个 Chrome / Edge 浏览器扩展，可以把当前打开的小红书帖子整理成素材包。

适合用来做运营拆解、设计参考、选题收集、商品灵感整理和个人资料归档。

## 你可以用它做什么

- 一键识别当前小红书帖子里的图片
- 批量保存选中的图片素材
- 保存帖子正文、原帖链接和当前页面截图
- 按帖子标题自动建立素材文件夹
- 默认全选素材，也可以单独选择 / 取消选择
- 保存路径默认是浏览器下载文件夹，也可以选择其他文件夹

## 仓库结构

```text
extension/   浏览器扩展文件，安装时选择这个文件夹
docs/        隐私说明、测试说明和补充文档
README.md    当前说明文件
```

安装扩展时，只需要使用 `extension/` 文件夹。

## 安装方法

### Chrome

1. 下载这个 GitHub 仓库。
2. 解压到本地文件夹。
3. 打开 Chrome，进入：

```text
chrome://extensions/
```

4. 打开右上角“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择项目里的 `extension/` 文件夹。
7. 浏览器右上角出现扩展图标后，就安装好了。

### Edge

1. 下载这个 GitHub 仓库。
2. 解压到本地文件夹。
3. 打开 Edge，进入：

```text
edge://extensions/
```

4. 打开“开发人员模式”。
5. 点击“加载解压缩的扩展”。
6. 选择项目里的 `extension/` 文件夹。

## 使用方法

1. 打开一个小红书帖子页面。
2. 等待帖子图片加载完成。
3. 点击浏览器右上角的“小红书图文素材助手”图标。
4. 插件会自动识别当前帖子里的素材。
5. 默认会选中全部素材，你也可以点击缩略图取消某些图片。
6. “保存路径”默认是浏览器下载文件夹，点击“选择”可以改成其他文件夹。
7. “存储为”默认使用帖子标题，也可以手动修改。
8. 点击“保存素材包”或“只保存图片”。

## 保存结果

保存完整素材包时，文件夹结构类似：

```text
帖子标题/
  images/
    01.jpg
    02.jpg
  videos/
    01.mp4
  正文.txt
  原帖链接.html
  截图.png
```

选择“只保存图片”时，只会保存图片文件。

## 常见问题

### 为什么没有识别到图片？

可以先确认：

- 当前页面是小红书帖子页面
- 图片已经加载完成
- 试着滚动帖子或切换轮播图
- 点击插件右上角的刷新按钮重新识别

### 为什么保存路径默认是下载文件夹？

浏览器扩展不能直接读取你的系统文件路径。默认情况下，素材会保存到浏览器下载文件夹。你也可以在插件弹窗里点击“选择”，授权保存到其他文件夹。

### 可以下载视频吗？

如果页面里能识别到可直接保存的 MP4 视频，保存完整素材包时会一起保存到 `videos/` 文件夹。部分页面的视频地址可能无法直接获取。

### 会上传我的数据吗？

不会。插件只在本地读取当前页面内容，并把素材保存到你选择的位置。详细说明见 [docs/PRIVACY.md](./docs/PRIVACY.md)。

## 注意事项

- 请只保存你有权使用的内容。
- 请勿将他人内容用于未授权搬运、商用或侵权用途。
- 小红书页面结构变化时，插件识别规则可能需要更新。
